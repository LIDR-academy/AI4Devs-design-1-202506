# Promt base 

Toma el rol de product manager experto en el diseño de soluciones web asistidas con funcionalidades automaticas y de inteligencia artificial.  Toma como referencia el modelo imagen 1713970482103.avif en este proyecto que define las etapas de la funcionalidad e un sistema ATS (Applicant-Tracking System), principios de diseño UI y UX,  web scraping y data extraction de acuerdo con las reglas configuradas en este proyecto, y buenas pacticas en el diseño de sistemas web y arquitectura de software. 

Toma en cuenta los siguientes enfoques funcionales: facilidad de uso, incluir lista de tareas generadas por componentes de inteligencia artificial, integración con platformas como correo electrónico, web scraping a portales de empleo, preselección y calificacon de candidatos con mayor afinidad a perfil requierido asistido por IA, trazabilidad de la información, integración con agenda de citas para entrevista. Las herraientas externas deben ser opensource e integrarse con google drive, gmail y officce 365.

Esto permitirá que la aplicación software sea lider en el mercado. por optimizar el tiempo de los reclutadores de personal y brindarles asistencia para ejeuctar sus labores de manera más eficaz y eficiente.

Desglosa las funcionalidades clave haciendo uso de esta información, plasmalo en el archivo LTI_DisenoAltoNivel e incluye un diagrama canvas de modelo de negocio y un diagrama conceptual de las funcionalidades y caracteristicas.
